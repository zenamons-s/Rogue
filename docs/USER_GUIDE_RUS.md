# Руководство пользователя Rogue (на русском)

## Как запустить игру
1. Убедитесь, что установлен Go (рекомендуется версия 1.20+).
2. В корне проекта выполните:

```bash
go run ./src/cmd/rogue
```

3. В стартовом приглашении выберите:
- `1` — продолжить игру из сохранения;
- `2` — начать новую игру;
- `q` — выйти.

## Управление
- `W/A/S/D` — движение
- `h` — открыть список оружия
- `j` — открыть список еды
- `k` — открыть список эликсиров
- `e` — открыть список свитков
- `t` — статистика прохождения
- `l` — таблица лучших попыток
- `q` — выход
- `?` или `i` — помощь

## Обозначения на карте
- `@` — игрок
- `z/v/g/O/s/e` — враги (разные типы противников)
- `f` — еда
- `p` — эликсир
- `r` — свиток
- `w` — оружие
- `$` — сокровище
- `>` — выход на следующий уровень
- `#` — стена
- `.` — пол комнаты
- `+` — коридор
- `D` — дверь

## Как использовать предметы
1. Нажмите клавишу категории (`h`, `j`, `k`, `e`).
2. Выберите номер предмета (`1..9`).
3. Для оружия можно нажать `0`, чтобы снять его.

Типы предметов:
- **Еда**: восстанавливает здоровье.
- **Эликсиры**: дают временные усиления (например, сила/ловкость/максимальное здоровье).
- **Свитки**: применяют мгновенные эффекты улучшения.
- **Оружие**: повышает боевые характеристики.

## От чего игрок умирает
- От атак врагов.
- Когда здоровье (`Здоровье`) падает до нуля.
- Временные эффекты могут косвенно повлиять на выживаемость, но поражение фиксируется при `HP <= 0`.

## Цель игры
- Исследовать подземелье и проходить этажи.
- Собирать сокровища и полезные предметы.
- Побеждать врагов и повышать счёт.
- На каждом уровне находить выход (`>`), чтобы перейти дальше.
- Игра считается выигранной при прохождении последнего уровня.

## Сохранение и загрузка
- Игра автоматически сохраняется по ходу прохождения.
- При следующем запуске можно выбрать продолжение (`1`) и загрузить последнее сохранение.
- Если сохранение отсутствует, будет запущена новая игра.

## Статистика и таблица лучших
- `t` — показывает текущую статистику попытки (уровень, сокровища, победы над врагами и т.д.).
- `l` — показывает таблицу лучших попыток.

## Как выйти из игры
- Во время игры нажмите `q`.
- На экране помощи выйти можно `Esc`, `Enter` или `q`.
